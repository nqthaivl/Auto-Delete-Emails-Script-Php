# Hướng Dẫn Cài Đặt Extension "Auto URL Runner"

Extension này giúp bạn tự động mở một đường dẫn (URL) theo chu kỳ thời gian cài đặt sẵn (Giờ, Phút, Giây).

## Cách cài đặt vào Chrome

1.  Mở trình duyệt Google Chrome.
2.  Truy cập vào địa chỉ: `chrome://extensions/`
3.  Bật chế độ **Developer mode** (Chế độ dành cho nhà phát triển) ở góc trên bên phải.
4.  Nhấn vào nút **Load unpacked** (Tiện ích đã giải nén).
5.  Chọn thư mục `extension` nằm trong thư mục dự án này (`c:\Users\Admin\Downloads\Del Email\extension`).
6.  Extension **Auto URL Runner** sẽ xuất hiện trên thanh công cụ.

## Cách sử dụng

1.  Nhấn vào biểu tượng Extension hình mảnh ghép hoặc icon mặc định trên thanh Toolbar.
2.  Nhập **Đường dẫn (URL)** bạn muốn chạy (Ví dụ đường dẫn đến file `delete_emails.php` trên localhost hoặc server).
3.  Cài đặt thời gian lặp lại (Giờ, Phút, Giây).
4.  Nhấn **LƯU & BẮT ĐẦU CHẠY**.
5.  Extension sẽ chạy ngầm và tự động mở tab mới theo đúng lịch trình bạn cài đặt.

## Lưu ý
-   Để extension hoạt động liên tục, bạn cần giữ trình duyệt Chrome mở.
-   Nếu bạn muốn dừng, mở lại extension và nhấn **DỪNG LẠI**.
